# api-node-memoria

Projeto desenvolvido para consolidar conceitos básicos de organização de um projeto e manipulação de dados em memória. Este repositório serve como base sólida para entender o fluxo de uma requisição antes da implementação de persistência com bases de dados SQL ou NoSQL.

## 🎯 Objetivos do Projeto

- Aplicar a separação de responsabilidades (SoC).
- Praticar o desenvolvimento de APIs RESTful utilizando Node.js.


## 🛠️ Tecnologias e Ferramentas

- **Runtime**: Node.js
- **Framework**: Express.js
- **Padronização**: JavaScript ES6+
- **Produtividade**: Nodemon para reinicialização automática em desenvolvimento.

## 🚀 Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Luizinho101/api-node-memoria.git
   ```
2. Entra na pasta do projeto
    ```bash
    cd api-node-memoria
    ```
3. Instale as dependências:
    ```bash
    npm install
    ```
4. Execute a aplicação:
    ```bash
    npm start
    ```
